# 微信小程序－题库

### 说明：

实现题库功能，特色：
- 判断题型和单选题型
- 做题进度和对错显示

### 数据接口:

- https://api.getweapp.com/vendor/com1217

### 目录结构：

- public — 存放项目公共库文件
- pages — 存放项目页面文件
- utils — 存放项目日期处理文件

### 开发环境：

微信web开发者工具 v0.11.112301
